STEPS TO INSTALL
1. Enable in admin/modules (extends)
2. Add 'Comment Alert Block' to section in admin/structure/block (Block Layout)
Done!
This modules uses cookies to track the number of comments on pages that a user decides to follow. An alert will be displayed if there is a new comment on a page.
Only users that comment can choose to get notifications from pages.
Users can unfollow pages and dismiss alerts.
